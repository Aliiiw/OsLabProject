# OsLabProject

OsLabProject is a small Java/IntelliJ project created for operating-systems lab work. The repository currently contains a basic Java entry point and the OS lab assignment PDF that documents the exercises for the course/lab.

## What Is Included

- A simple Java source folder under `osLabProject/src`.
- `Main.java` as the current application entry point.
- IntelliJ IDEA module configuration.
- `OS-lab assignments.pdf` as the assignment reference document.
- Compiled output generated by IntelliJ under `out/production`.

## Project Structure

```text
.
├── OS-lab assignments.pdf
├── README.md
└── osLabProject
    ├── osLabProject.iml
    ├── src
    │   └── Main.java
    └── out/production/osLabProject
        └── Main.class
```

## Current Code

The current Java entry point prints a simple `Hello world!` message. The repository appears to be an initial workspace for OS lab assignments rather than a completed operating-systems simulator or tool.

## Getting Started

1. Clone the repository.
2. Open the `osLabProject` folder in IntelliJ IDEA.
3. Make sure a JDK is configured for the project.
4. Run `Main.java`.

## Manual Compile

```bash
cd osLabProject
javac src/Main.java
java -cp src Main
```

## Notes

Use the included PDF as the reference for the lab tasks. As the assignments are implemented, the README can be expanded with each exercise, expected inputs/outputs, and run instructions.
